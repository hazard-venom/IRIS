# IRIS

IRIS is meant to be used with the client code running on a Raspberry Pi.

## Hardware Needed

- Raspberry Pi
- Microphone
- Camera module
- Earpiece or audio output device

## How To Use

Use the client code on a Raspberry Pi that has:

- a connected mic for voice input
- a camera module for visual input
- an earpiece for audio responses

The main client files in this project are:

- `iris_client.py`
- `new.py` ## server code 


Install the required dependencies first, then run the client code on the Pi with the hardware connected and configured.
